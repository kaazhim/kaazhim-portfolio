# Portfolio Change Requests

Use this folder to write anything you want to change in the portfolio later.

Suggested format:

```md
## Change title
- Section: Home / Infra Lab / Projects / Skills / Resume / Contact
- What to change:
- New text or idea:
- Priority: Low / Medium / High
```

Current live preview:

http://127.0.0.1:5173/

Main files:

- `src/App.jsx` - page sections and interactions
- `src/data.js` - resume/profile/projects/dashboard content
- `src/styles.css` - colors, layout, animation, responsive design
- `public/assets/` - images, resume preview, QR code, project assets
